## Description
Implement A* pathfinding on the arena tile grid as defined in architecture.md section 3.3.

## Requirements
- `pathfinder.gd` — A* pathfinding on the tile grid
- Uses arena_manager to query walkable tiles
- Recalculates path every 10 ticks (not every tick — performance)
- Walls and pillars are impassable; cover blocks allow movement through
- Brotts cannot overlap (CharacterBody2D collision)
- Returns path as Array of tile positions
- Chassis speed affects movement along path (Scout 200, Brawler 120, Fortress 70 px/sec)

## Acceptance Criteria
- [ ] A* finds shortest path between two tile positions
- [ ] Walls and pillars correctly blocked
- [ ] Path recalculation every 10 ticks
- [ ] Movement speed respects chassis data
- [ ] Handles no-path-found gracefully (Brott holds position)
- [ ] Unit tests: path through open field, around walls, no-path scenario
- [ ] Clean GDScript following architecture patterns

## Reference
- docs/architecture.md section 3.3
- Tile size: 32×32 px
- Recalc interval: every 10 ticks
