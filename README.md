# 🤖⚔️ BattleBrotts

> Build. Program. Fight.

Build autonomous combat Brotts by choosing a chassis, bolting on weapons and armor, then teaching them how to fight with drag-and-drop Behavior Cards. Watch your creation battle it out in top-down arenas where strategy beats stats.

**Engine:** Godot 4 | **Platform:** Web (HTML5) | **Genre:** Auto-battler / Strategy
